Flexbox Cheatsheet
property Use
Flex-Container The parent element you activate display:flex on
Display:flex To activate flexbox on the parent container
Display: inline-flex To activate inline-flex on the parent container
Flex items The children elements in the parent container
Flex-direction To control the direction in which flex-items are laid along the main axis
Flex-wrap This allows flex-items force themselves into a line or wrap unto multiple lines
Flex-flow Shorthand property for flex-direction and flex-wrap values
Justify-content This defines how flex-items are laid out on the main axis
Align-items This defines how flex-items are laid out on the cross axis
Flex-grow,Flex-shrink Controls how much a flex item should grow or shrink
Flex-basis Specifies the initial size of a flex-item
Flex Shorthand used to set the flex-grow,flex-shrink,flex-basis properties all at once
